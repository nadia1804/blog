Jurnal pembelajaran gagal divalidasi : Pdf tidak terbaca

Deddy Hidayat 29 Juni 2025
Bagikan ke :

Jurnal ditolak dengan alasan dokumen tidak terbaca, padahal revisi berkali-kali? tetap ditolak, bagaimana cara memperbaikinya? 

Alasan Jurnal pembelajaran pdf tidak terbaca

Ada banyak alasan kenapa jurnal pembelajaran tidak terbaca,

  1. Dokumen pdf rusak
  2. Dokumen pdf di proteksi
  3. Dokumen pdf berbentuk gambar

Cara memperbaiki jurnal yang tidak terbaca

1. Save as pdf langsung dari ms word

ini adalah saran perbaikan saat anda mendapatkan notifikasi dokumen tidak terbaca :

Dokumen tidak terbaca, Anda dapat mengupload PDF dari ms.word agar dokumen mudah terbaca

Caranya memperbaikinya, pada Ms word :

File > simpan sebagai > pilih lokasi penyimpanan > pilih format .pdf

atau

File > Export > Buat PDF

Caranya memperbaikinya, pada di Canva :

Bagikan > unduh > pilih jenis formatnya .PDF standar > download

seharusnya cara ini sudah bisa agar dokumen terbaca oleh sistem, lalu bagaimana jika belum bisa terbaca?

2. Edit ulang struktur dokumen dan kurangi gambar

Jika dokumen masih belum terbaca padahal sudah berbentuk pdf dan mengikuti cara no 1, kita perlu melihat isi dokumen, ada kemungkinan dokumen berisi banyak elemen gambar.

Jurnal yang anda buat bisa mengandung :

  1. rubrik (hasil scan tulisan tangan)
  2. foto dokumentasi (foto)
  3. umpan balik teman sejawat (hasil scan tulisan tangan)
  4. dan rifleksi (hasil scan tulisan tangan)

Elemen gambar bukan cara yang baik untuk menyajikan jurnal karena sistem tidak dapat membaca gambar, 

Foto/gambar hasil scan tulisan tangan memang diperlukan namun jangan mendominasi seluruh isi dokumen.

caranya perbaikannya :

  • edit ulang dokumen anda
  • rubah hasil “tulis tangan yang anda scan” menjadi bentuk “ketikan”.

seharusnya cara ini sudah bisa agar dokumen terbaca oleh sistem, lalu bagaimana jika belum bisa terbaca?

3. Revisi di sisi server

Terkadang kesalahan bukan berasal dari anda. kita sudah yakin dokumen sudah benar, kenapa masih belum terbaca? Masalanya ada di sisi server. dimana server masih menyimpan cache data lama anda.

caranya :

  • hapus topik jurnal pembelajaran yang telah anda pilih 
  • kemudian pilih lagi topik yang sama
  • isi formulirnya judul, pertanyaan
  • kemudian upload ulang dokumen

cara ini memakan waktu 4 hari untuk divalidasi karena di anggap sebagai jurnal baru, bukan di anggap merevisi dokumen.

Kesimpulan

Artikel ini ditulis berdasarkan pengalaman pribadi, "saat itu jurnal pembelajaran saya ditolak dengan alasan tidak terbaca". saya lakukan revisi hingga 4 kali dengan berbagai metode, dari cara nomor 1, tehnik OCR PDF, dan cara nomor 2, namun masih ditolak.

setelah saya analisa permasalahannya bukan pada dokumen, namun masalah nya ada pada sisi server. sehingga mungkin artikel ini akan bermanfaat untuk anda.
